# C_Code - 嵌入式C语言系统化学习工程

## 项目简介

本项目是一个面向嵌入式开发的 C 语言系统化学习工程，涵盖了 C 语言的核心知识点，专为嵌入式开发学习设计。所有代码遵循 MISRA-C:2012 编码规范，适合作为嵌入式开发的学习参考和代码模板。

## 功能特性

- **11 个学习模块**：涵盖 C 语言核心知识点
- **静态内存分配**：数据结构模块采用静态内存，适合资源受限环境
- **完善的错误处理**：统一的错误码和错误处理宏
- **调试工具**：日志系统、断言、十六进制转储等
- **测试框架**：简化测试用例编写的宏定义

## 编译与运行

### 环境要求

- Windows 10/11
- Visual Studio 2019 或更高版本（需要 v145 工具集）

### 编译步骤

1. 使用 Visual Studio 打开 `C_Code.sln`
2. 选择构建配置（Debug 或 Release）
3. 选择平台（x64 推荐）
4. 按 `Ctrl+Shift+B` 编译项目

### 运行方式

**交互模式**：
```bash
# 直接运行，通过菜单选择模块
C_Code.exe
```

**命令行模式**：
```bash
# 运行所有模块演示
C_Code.exe --demo

# 运行指定模块演示（1-11）
C_Code.exe -d 5

# 显示帮助信息
C_Code.exe --help
```

## 代码规范

本项目遵循 **MISRA-C:2012** 编码规范，主要包括：

- 使用固定宽度整数类型（`uint32_t`、`int16_t` 等）
- 所有函数都有返回值和参数检查
- 避免使用全局变量
- 使用 `static` 限制函数和变量的作用域
- 所有宏定义使用大写命名
- 详细的 Doxygen 风格注释

## 公共模块 API

### 常用宏定义

```c
// 数组大小
size_t count = ARRAY_SIZE(arr);

// 最值和绝对值
int max_val = MAX(a, b);
int min_val = MIN(a, b);
int abs_val = ABS(x);

// 位操作
BIT_SET(reg, 3);      // 设置第3位
BIT_CLEAR(reg, 1);    // 清除第1位
BIT_TOGGLE(reg, 5);   // 翻转第5位
bool is_set = BIT_CHECK(reg, 2);  // 检查第2位

// 字节操作
uint8_t lo = LOW_BYTE(0x1234);    // 0x34
uint8_t hi = HIGH_BYTE(0x1234);   // 0x12
```

### 日志系统

```c
// 设置日志级别
log_set_level(LOG_LEVEL_INFO);

// 输出日志
LOG_DEBUG("调试信息: %d", value);
LOG_INFO("系统启动");
LOG_WARNING("内存不足");
LOG_ERROR("打开文件失败: %s", filename);
LOG_FATAL("致命错误");
```

### 测试框架

```c
// 测试函数示例
int32_t test_my_function(void) {
    TEST_ASSERT(my_function() == 0, "my_function 应返回 0");
    TEST_ASSERT_EQUAL(42, calculate(), "计算结果应为 42");
    return 0;  // 成功
}

// 运行测试
RUN_TEST(test_my_function);
```

## 学习建议

1. **按顺序学习**：模块按知识依赖关系排列，建议从模块 01 开始
2. **动手实践**：每个模块都有演示函数，运行并观察输出
3. **修改代码**：尝试修改演示代码，加深理解
4. **编写测试**：使用测试框架编写自己的测试用例
5. **阅读源码**：代码注释详细，阅读源码是最好的学习方式
